# 🟩 DTexturePack — Webowy Generator Paczek Tekstur (Minecraft)

Strona **DTexturePack** to webowe narzędzie do prostego tworzenia paczek tekstur do **Minecrafta (Java & Bedrock)** — działa w przeglądarce i nie wymaga instalacji.  
👉 [https://dtexturepack.danielo535.pl](https://dtexturepack.danielo535.pl)

> 📌 Projekt jest w **BETA** — mogą pojawiać się błędy lub zmiany funkcji.

---

## 🚀 Co to daje?

DTexturePack pozwala:  
- ustawić ikonę i nazwę paczki,  
- wybrać platformę (Java / Bedrock / Geyser),  
- dodawać tekstury przedmiotów, bloków i akcesoriów,  
- wygenerować paczkę gotową do pobrania 💾.

---

## 🧠 Jak korzystać

1. Wejdź na: [https://dtexturepack.danielo535.pl](https://dtexturepack.danielo535.pl)  
2. Załaduj projekt lub ZIP (jeśli masz już paczkę)  
3. Uzupełnij **Pack Settings**:  
   - ikona paczki (PNG)  
   - nazwa paczki  
   - opis paczki  
   - wersja Minecraft  
   - typ platformy (Java / Bedrock / Geyser)  
4. Dodaj swoje tekstury (Items, Blocks, Armor, Text)  
5. Kliknij **Generate Pack** i pobierz paczkę
